# デザインガイドライン

デザインの正は実装(`app/src/`)。本ファイルは方針とルールのみ。

## 方針

- UIの質が競争力の核。「リンクを踏んだ瞬間にInstagramと違う」という体験が重要
- 作品が大きく、余白を持って表示される
- 展覧会のタイトル・期間・場所などのメタ情報がきれいにまとまっている
- スマホで崩れない
- **団体 → 展覧会 → 作品**という階層構造が見えることがSNSの時系列フィードとの根本的な差別化(「この展覧会のすべての作品を見る」が自然にできる)

## カラーパレット(実装は `app/src/lib/tokens.js` の `T`)

ベースは**紙色・墨色**の2階調。アクセントは次の2系統に限定する。

| 役割 | 用途 | メモ |
|------|------|------|
| **Accent(テラコッタ)** | 主要 CTA、リンク強調、閉じる UI のアクセント枠 | `T.accent` / `T.accentInk` |
| **Gold(黄土〜金)** | ヘッダー/ナビの区切り、強調シャドウ、副バッジ | `T.gold` |

プレースホルダ背景などのムード色は**寒色(水色・紺の装飾)より暖色ベージュ**(`T.surfaceMuted`)を使う。装飾スポットが増えすぎないよう、ミント・すみれ等のサブカラーは基本追加しない。

## モバイル(情報密度・タップ)

- 一覧・ヘッダー・ボトムナビは、縦スペースを詰めつつ**主要タップ領域はおおまかに 44×44px 以上**(アイコンボタンは `minWidth` / `minHeight` で確保)
- 見出し・カード行のフォントはデスクトップより一段階小さくしてよい(例: 団体一覧タイトル 22px 前後)

## 外部リンク(団体ページ)

- 右上矢印だけにせず、**🔗 + ラベル + 下線付きホスト名**(またはカード左ボーダー)で「外部へ行くリンク」と分かるようにする
- 公開サイト内の内部リンクとは、`T.accent` の下線・左アクセントで差別化する

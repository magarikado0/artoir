# デザインガイドライン

## 方針

- UIの質が競争力の核。「リンクを踏んだ瞬間にInstagramと違う」という体験が重要
- 作品が大きく、余白を持って表示される
- 展覧会のタイトル・期間・場所などのメタ情報がきれいにまとまっている
- スマホで崩れない

## Instagramとの差別化ポイント

1. **団体 → 展覧会 → 作品** という階層構造が見える
   - SNSの時系列フィードとは根本的に違う体験
   - 「この展覧会のすべての作品を見る」が自然にできる

2. **背景色カスタマイズ**
   - 作品一覧ページの背景色を団体・展覧会ごとに変えられる
   - 書道部なら和紙調、写真部なら黒背景、など世界観を演出できる

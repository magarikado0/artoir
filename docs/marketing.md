# マーケティング — ターゲット・広報・アウトリーチ

> 最終更新: 2026-07-05(旧 `marketing/target.md` / `messaging.md` / `promotion.md` / `outreach.md` を統合)

## ターゲットユーザー

- 大学のサークル・部活(書道部・写真部・美術系など)
- 個人作家・アーティスト
- 小規模ギャラリー・団体

初期ユーザーは「すでに発信意欲がある団体」に絞る。チームは3人(開発主体2人)。

## メッセージング

コアメッセージ: **「展覧会をリンク1本で共有する」**

- 使用シーン: 展覧会中〜直後(来場できなかった人に見せる、記録として残す)、オンライン展覧会(会場を持たずにArtoirだけで開催)
- SNSで告知 → Artoirのリンクで詳細を見る、という共存設計。展覧会前の作品公開はネタバレになるため、告知用途はメタ情報(日時・場所)のみになる想定

## 広報方針

> 2026-07 の検討記録。ステータス: **提案**(チーム合意前。決定したら本注記を外す)。

Artoirの自然な導線は「来場者・SNSフォロワーが展覧会ページを見る」ことで、**その閲覧者の中に次の主催者(他サークルの部員)がいる**。広告を打つのではなく、公開ページ自体を獲得チャネルにする。

### 施策

1. **京大書展の事例化(最優先)** — 書展(2026-06-21)の閲覧数・シェア経路・部員/来場者の反応を1枚の事例資料にまとめる。「○○人が閲覧、うち会期後の閲覧が○%」のような数字が営業の武器になる。閲覧数を計測していなかった場合、次の展示までに最低限のアクセス計測を入れるのが先決
2. **公開ページに「Powered by Artoir」CTA** — フッターに控えめな「Powered by Artoir — あなたの展覧会も公開できます」を設置。Linktree型の自己増殖ループ。ロゴ表示は制約ではなく**広告枠**と捉える
3. **季節性に合わせて動く** — サークル展覧会の山は**11月の学園祭と春**。7〜9月が「秋の学祭展示で使ってもらう」勧誘の適期。学園祭は**収益源ではなく広報の季節**として、芸術系サークルに個別に声をかける(学祭実行委への一括販売は「芸術のアーカイブ」の建付けに合わないため却下済み・2026-07)
4. **Artoir公式SNS** — 掲載許可を取った展覧会を公式アカウントで紹介。団体側にもメリットがあり営業トークの材料にもなる。低コストで効く

### 公式Instagram運用方針

> 2026-07-05 の検討記録。ステータス: **提案**(チーム合意前。決定したら本注記を外す)。施策④「Artoir公式SNS」の具体化。

#### 基本設計

- **毎日フィード投稿はしない**。コンテンツ源が現状書展1回分しかなく、3人チームでは質が持たない。フィード週2〜3本+ストーリーズで毎日接触が現実的ライン
- DM営業を先行するが、**DMを送ると相手はプロフィールを見に来る**ため、営業開始前に10〜15投稿の「受け皿」を作り置きしてから公開する
- 秋学祭で複数団体が載り作品ストックが貯まったら、フィード毎日化(1日1作品)を検討

#### コンテンツの柱

| 柱 | 素材 | 備考 |
|---|---|---|
| 展覧会紹介のキュレーション | 他団体の開催予定展(公開告知情報) | 最重要。下記「DMの口実」参照 |
| 1日1作品 / 書展の振り返り | 京大書展 | SNS掲載同意の確認が前提(下記) |
| プロダクト紹介 | 使い方カルーセル、画面収録(Reels)、開発ログ | 許可不要の自前素材。「学生が作っている」ストーリーが部員層に刺さる |
| コンセプト系テキストグラフィック | 「展覧会は1週間で終わる。作品はずっと残せる」等 | テンプレ1枚で量産、グリッドの間埋め |

#### キュレーション投稿 = 営業の口実(中核アイデア)

他大学・他サークルの開催予定展を紹介する投稿を柱にする。「学生の展覧会を紹介するアカウントです。御団体の展示を紹介させてもらえませんか?」というDMは営業DMより返信されやすく、承諾を得ればフライヤー画像の掲載許可も同時にクリアできる。紹介で関係を作ってから「会期後に作品をアーカイブしませんか」と二段目の営業につなげる。**投稿活動自体が営業活動になる**設計で、「公開ページ自体を獲得チャネルにする」思想のInstagram版。

#### 運用の型

- プロフィールのリンクは artoir.net 固定。bioにコアメッセージ「展覧会をリンク1本で共有する」
- ストーリーズのリンクスタンプで展覧会ページを直貼り(インスタで唯一クリックさせられる場所)
- 作品写真は正方形+余白の統一テンプレ(Canva等)。グリッドが揃うと「アーカイブ」らしさが出る
- ハッシュタグはターゲットが見るタグに絞る: #書道 #大学書道 #学生書道 #展覧会 #美術部 #写真部 など
- 紹介する団体・作者は必ずタグ付け。団体側のリポストが最大の拡散経路

#### 注意: 掲載許可はSNS転載をカバーしていない

書展アンケートの同意は「Webサイト Artoir にて公開」のみで、Instagram転載は許可範囲外と読むのが安全。次回アンケートに「Artoir公式SNSでの紹介」への同意を追加し、書展出品者への遡及確認も検討する(図録利用同意の未取得と同じ構図)。

#### 立ち上げ手順

1. 書展素材+プロダクト紹介で初期投稿を10本ほど作り置き(SNS掲載同意の確認と並行)
2. アカウント公開、週2〜3で投稿開始
3. 京大内サークルへDM/直接声かけ(8〜9月の予定通り)。「公式インスタで紹介します」を営業材料に使う
4. 並行して他団体への展覧会紹介DMを回し、素材と関係を増やす

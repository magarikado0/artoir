# マネタイズ

## 方針(2026-07改訂)

主催者向けの月額サブスクは棚上げし、**図録PDF生成の展覧会単位課金**を本命とする。

- Web公開(展覧会ページ・作品掲載・シェアリンク)は**点数無制限・完全無料**
- 収益は「展覧会にすでに流れているお金(印刷費)の通り道になる」ことで得る

### なぜサブスクをやめるか

- サークルは年1〜2回しか展覧会をやらず、月額課金は支出構造に合わない(部費は「イベント単位」で支出される)
- フリーミアムの一般的な転換率(2〜5%)では、目標有料数に対して母数が全く足りない
- 閲覧者とデータが増えるほど図録の価値が上がるため、無料側を制限する理由がない

### なぜ図録か

図録づくりの本当の苦痛は印刷ではなく編集作業(数十点の作品画像・タイトル・キャプションの収集と組版)。Artoirは展覧会ページ作成の副産物としてこれを構造化済みなので、「展覧会ページを作ったら図録が出てくる」は他ツールで代替しにくい。

DM・作品ポストカードは検討の結果見送り(使うデータが少なく、Canvaやネット印刷直接入稿で代替可能なためArtoirの優位性がない)。

## 商品形態

**印刷は扱わず、入稿データ(PDF)の生成だけを売る。**

| 項目 | 内容 |
|------|------|
| 商品 | 展覧会ページから印刷用図録PDFを自動生成(表紙・目次・作品ページ・奥付) |
| 価格 | 1展覧会あたり2,000〜3,000円の買い切り(初回はモニター価格で検証) |
| 印刷 | 主催者自身がPDFをネット印刷(ラクスル・グラフィック等)に入稿 |
| 決済 | Stripeの決済リンク |

在庫・発送・印刷品質のクレームを抱えない。印刷発注の代行(マージン取得)は需要確認後の拡張とする。

## 検証計画

1. 京大書展(2026年6月)のデータから図録PDFを試作
2. 書道部に見せ、**次回展示で実額請求**して支払い意思を検証(「払うか聞く」は社交辞令になるため、実際に売る)
3. 秋の学祭シーズン(2026年11月)に他サークルへ「図録も作れます」で営業 — 試作と反応確認は夏中に完了させる

## 目標

- 有償図録で月1〜2万円のランニングコストを賄う
- 初期は赤字埋め程度と割り切り、不足分は助成金・学生起業支援の活用も検討する

## 検討した代替案の記録(2026-07)

アイデアの判定基準は4つ:**誰の財布か/その支出はすでに存在するか/Artoirの構造化データが手作業を肩代わりするか/「芸術のアーカイブ」という建付けに合うか**。

### 将来の検討候補(時期尚早として保留)

| 案 | 内容 | 保留理由・再開条件 |
|----|------|-------------------|
| 貸しギャラリー提携 | ギャラリーが「展示プランにオンライン掲載ページ付き」を自分の客に売り、Artoirはギャラリーから月額/件数課金。図録の次のレバー筆頭 | 事業者相手なので月額が成立し、個人作家が自動流入する。ただし営業・信頼構築が必要。個人作家ユーザーが見え始めてから |
| 公募展・コンペのホスティング | 企業・自治体主催の公募展の応募受付+掲載+結果発表をパッケージ販売。主催者予算は大きい | 応募管理・審査機能などほぼ別プロダクトの開発が要る。美術・書道系コンペに限れば建付けに反しない |
| 印刷アフィリエイト | 図録PDFの入稿先(ラクスル等)の紹介プログラムを踏む | 単価は小さいが図録機能に乗せるだけ。図録実装時に追加 |
| 作品売買の仲介手数料 | 原画販売の10〜20% | 決済・配送・トラブル対応の信頼インフラが必要。個人作家が増えたフェーズの話 |
| 画材メーカー等の協賛・広告 | 美術系学生に届く媒体としての広告価値 | 月数万PV超えてから。今は時期尚早 |
| 助成金・学生起業支援 | 京大インキュベーション、文化系助成等でランニングコストを賄う | マネタイズではなく「つなぎ」。初期は現実的な選択肢 |

### 見送り(理由つき)

| 案 | 見送り理由 |
|----|-----------|
| 主催者向け月額サブスク(旧方針: 月500円) | サークルは年1〜2回しか展覧会をせず月額は支出構造に合わない。転換率2〜5%では必要な母数(400〜2,000団体)が非現実的。「Webサービス月額費」という支出項目がサークル会計に存在しない |
| 広告用DM生成 | 1枚モノで載る情報が少なくCanvaで代替可能。Artoirの構造化データの優位性が出ない |
| 作品ポストカード(物販/主催者まとめ買い) | 使うデータが画像1枚でネット印刷直接入稿と大差ない。優位性が出ない |
| 学祭実行委への一括アーカイブ販売 | 実行委は予算を持つが、模擬店・技術展示等が混ざり「芸術のアーカイブ」の建付けに反する。学祭は広報の季節として使う(`../marketing/promotion.md`) |
| 投げ銭 | 閲覧規模が小さいうちは決済手数料に食われる |
| OB/OG向けアーカイブ課金 | 支払い動線が複雑な割に単価が低い |
| 受託制作(ギャラリー特注サイト) | お金にはなるがプロダクトが進まず、学業との両立も苦しい |

## 未決事項

- **同意と作者名**: 図録(印刷・販売物)への利用は現行のオンライン公開同意と別途必要。また図録には作者名を載せる慣行があり、「作者名を作品に紐づけない」方針との整理が必要(「Webは匿名・図録は同意者のみ記名」等)。次回の出品調査アンケートまでに決める
- **画像解像度**: 印刷は300dpi必要。元画像を原寸でStorageに保持する方針の確認
- **収益の受け皿**: 受取名義(個人/任意団体)、特商法表記、チーム内の分配ルール
- **支払い手段**: 払うのはサークルの会計(部費)なので**領収書発行は必須**。可能なら請求書払い(Stripe invoice)も視野に

# 機能一覧

> 最終更新: 2026-07-05（実装との突き合わせで全面改訂）。詳細な仕様は `docs/spec.md`。

## 実装済み

### 公開ページ

- トップ: 全展覧会一覧（検索付き）
- 団体一覧・団体ページ（名前・説明・SNS/HPリンク・展覧会一覧）
- プロフィールページ（作品・所属団体・個人の展覧会。`/@{slug}` エイリアスあり）
- 展覧会ページ（タイトル・会期・場所・説明・作品ギャラリー・作品モーダル）
- シェアリンク（公開ページはすべて認証なしで閲覧可）

### 作品

紐づく情報: タイトル・説明文・画像・作者（`artwork_creators` で紐づけ、表示ON/OFF可）

※ 旧方針「作者名は掲載しない」は変更済み。作者は本人・団体の設定で**表示するかどうかを選べる**形になった。

### アカウント・管理

- ログイン（Google OAuth / メール+パスワード）
- プロフィール設定、団体作成、メンバー管理（owner / admin）
- ダッシュボード: 展覧会の作成・編集・削除、作品のアップロード（自動圧縮）・並べ替え・作者設定

### お気に入り

- 作品・展覧会・団体・プロフィールをブックマーク保存、`/collection` で一覧（保存数は非公開）

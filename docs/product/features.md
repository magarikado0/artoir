# 機能一覧

## MVP（最初にリリースする範囲）

### 団体ページ

表示する情報：
- 団体名
- 説明文
- SNSリンク（Instagram、X等）
- ホームページリンク
- 展覧会一覧

### 展覧会ページ

表示する情報：
- タイトル
- 開催期間・場所
- 説明文
- 作品一覧

カスタマイズ：
- 作品一覧の背景色を変更できる（団体ごとの世界観を出す）

### 作品

紐づく情報：
- タイトル
- 説明文
- 画像

※作者名は**掲載しない**（プライバシー配慮）

### URL設計

```
artoir.net/{org-slug}/exhibition/{exhibition-slug}
```

例：`artoir.net/kyodai-shodo/exhibition/2026-spring`

### シェアリンク

展覧会URLをそのままSNSで共有できる。

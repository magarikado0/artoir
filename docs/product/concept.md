# コンセプト

展覧会ごとに作品をまとめて公開できるポータルサイト。

**建付けは「芸術（作品）のアーカイブ」**。展覧会なら何でも載せるのではなく、芸術作品に限る（作品が映える世界観が価値の中核のため）。機能・マネタイズのアイデアは「載るものが芸術作品か」を先に通す。

## Problem

InstagramなどのSNSは時系列フィードしかなく、「展覧会」という単位でコンテンツを整理できない。展覧会ごとにアカウントを作る運用は手間がかかる。BehanceはあくまでBehanceは個人ポートフォリオ用で、複数人が参加した展覧会単位の整理には向いていない。

## Core Value

展覧会ページを作成してリンクを発行し、SNSで共有できる形にする。リンクを踏むと：

- UIが綺麗で作品が映える
- 展覧会ごとに作品がまとまって見られる
- その団体の過去・未来の展覧会一覧も見られる

SNSを置き換えるのではなく、**SNSで告知 → Artoirのリンクで詳細を見る**という共存設計。乗り換えコストがほぼゼロ。
